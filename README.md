# 📄 Empirical Economics Master Thesis LaTeX Template

This repository provides a **highly detailed LaTeX template** for writing a master’s thesis (As well as bachelor thesis a/o seminars) in **empirical economics**. It is designed to offer both a ready-to-use format and full transparency for those who want to understand or customize the underlying structure.

---

## 📦 Repository Structure

The repository is organized to serve two types of users:

### 1. 🚀 Quick Start (Ready-to-Use)

- **ZIP file**  
  Contains the complete template bundled for immediate use.  
  👉 Ideal if you just want to start writing your thesis without dealing with LaTeX structure.

- **PDF preview**  
  Shows how the final document looks.  
  👉 Useful for quickly evaluating layout, formatting, and structure.

---

### 2. 🛠️ Template Source (LaTeX Structure)

Inside the `template/` folder:

- Each **chapter is provided as a separate `.tex` file**
- Corresponding **`.bib` file(s)** included
- Fully modular structure for:
  - Easy customization
  - Clear separation of content
  - Scalable document management

👉 Ideal for users who want to:
- Learn from the template structure  
- Modify formatting or layout  
- Adapt it to their own university requirements  

---

## ✨ Features

- Clean and professional formatting
- Modular chapter structure
- Pre-configured bibliography setup
- Suitable for empirical research (tables, figures, references)
- Easy to extend and customize

---

## 🧑‍💻 How to Use

### Option 1: Simple Use
1. Download the ZIP file  
2. Extract it  
3. Open in your preferred LaTeX editor (e.g., Overleaf, TeXstudio)  
4. Start writing  

### Option 2: Advanced Use
1. Navigate to the `template/` folder  
2. Explore individual `.tex` and `.bib` files  
3. Customize structure, packages, or formatting as needed  

---

## 📌 Requirements

- LaTeX distribution (e.g., TeX Live, MiKTeX)  
- Recommended: Overleaf for easy collaboration  

---

## 📄 Notes

- The template is designed for **empirical economics theses**, but can be easily adapted to related fields.
- Always check your university’s formatting guidelines before submission.
